title: Gamified Silk Screen Cleaner
description:  The Gamified Silk Screen Cleaner is an assistive technology device developed in partnership with Spectrum Designs, a neurodiverse employment nonprofit, to improve the screen printing ink removal process for workers with disabilities. The system combines a motorized brush mechanism with a water-based cleaning solution to reduce manual labor and chemical exposure during screen cleaning. A gamified software component built on a Raspberry Pi adds a stopwatch, leaderboard, and point system to make the repetitive cleaning task more engaging. Design process features multiple mechanical iterations, including guide rail systems, secondary squeegee cleaner, and an initial handheld model.
